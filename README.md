# tiled
Go library for tiled

## tmx features
- [x] Deserialization of TMX xml format
- [x] Serialization of TMX xml format
- [ ] Custom Binary Serialization using JAMS
- [ ] Template File Support
- [ ] Wang Set Support

- [x] Object Layer Rendering
  - [x] Color Support

  - [x] Ellipse Rendering
